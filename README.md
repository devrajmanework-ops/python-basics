# Python Security Labs

This repository contains structured, hands-on Python scripts and mini-projects
built while learning Python with a focus on cybersecurity, automation,
and problem-solving.

The goal of this repository is not theory, but practical implementation
of core Python concepts that are commonly used in security tooling,
scripting, and automation.

---

## Repository Structure

```text
basics/        -> Core Python fundamentals (loops, conditions, input handling)
core_python/   -> Functions, error handling, OOP concepts
networking/    -> Socket programming and network-based scripts
web/           -> Web-related automation and attack simulations (lab-based)
crypto/        -> Hashing, encoding, and password-related utilities
tools/         -> Small CLI tools and utilities
